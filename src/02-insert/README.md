# 02-insert

This practice is orientated on learning the basics of the **inserts** using the `INSERT INTO` query. The script inserts different values into the users, companies and jobs tables. After the insertions, the script shows the final tables with `SELECT` queries.

---

## Concepts practiced

- Inserting data
- Select columns to insert data
- Tests with incorrect data types

---

## Usage

``` bash
node insert.ts
