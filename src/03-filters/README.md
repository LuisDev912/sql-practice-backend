# 03-filters

This practice is orientated on learning the basics of the **filters** using the `SELECT` query with `FROM` and `WHERE` clauses. This script also shows different ways of filtering data using `LIMIT` and `OFFSET`, `ORDER BY`, `BETWEEN`, etc.

---

## Concepts practiced

- Selecting data
- Filtering data
- Sorting data
- Managing received data

---

## Usage

``` bash
node filters.ts
