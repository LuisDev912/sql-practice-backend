# sql-practice-backend
A guided playground to practice the **SQL fundamentals** with small exercises.

---

## Goals

- Understand how to make SQL queries.
- Learn how to protect data.
- Work with databases using TypeScript.

---

## Tech Stack

- SQLite
- TypeScript

---

## Project Structure

Each folder contains a focused practice with its own README and executable code.

---

## How to Run
1. Clone the `[repository](https://github.com/LuisDev912/sql-practice-backend.git)`.
2. Navigate to any practice folder.
3. Run `node index.ts`
